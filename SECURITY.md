# Security Policy

Do not open a public issue for:

- exploit details against live infrastructure
- leaked credentials or signing keys
- anything that exposes private provider-account details
- private security findings against deployed ClawCombinator surfaces

## Reporting

Use the ClawCombinator contact surface and prefix the subject with
`Security report`:

- https://clawcombinator.ai/contact.html

Keep public bug reports high level if they touch security-sensitive behavior.
